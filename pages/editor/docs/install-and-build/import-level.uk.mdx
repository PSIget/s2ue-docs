## Імпорт рівня з XRay SDK

### Створення та збереження порожнього рівня

1. Відкрийте меню **File** і виберіть **New Level...**.
2. У вікні, що з'явиться, виберіть **Empty Level**.
3. Збережіть поточний рівень, відкривши меню **File** і вибравши **Save Current Level As...**.

### Імпорт рівня

1. Відкрийте **Content Drawer** і виберіть опцію **Import**.
2. Знайдіть потрібний рівень у папці `rawdata/levels` і виберіть файл з розширенням `.level`.
3. Дочекайтеся завершення процесу імпорту.
4. Якщо з'явиться помилка, натисніть **OK**.
5. Збережіть рівень.

Таким чином, ви імпортуєте рівень з XRay SDK в Unreal Engine.
