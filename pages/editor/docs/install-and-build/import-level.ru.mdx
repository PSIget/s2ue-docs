## Импорт уровня из XRay SDK

### Создание и сохранение пустого уровня

1. Откройте меню **File** и выберите **New Level...**.
2. В появившемся окне выберите **Empty Level**.
3. Сохраните текущий уровень открыв меню **File** и выбрав **Save Current Level As...**.

### Импорт уровня

1. Откройте **Content Drawer** и выберите опцию **Import**.
2. Найдите нужный уровень в папке `rawdata/levels` и выберите файл с расширением `.level`.
3. Дождитесь завершения процесса импорта.
4. Если появится ошибка, нажмите **OK**.
5. Сохраните уровень.

Таким образом, вы импортируете уровень из XRay SDK в Unreal Engine.
